# Sensor Comparison for Autonomous Robot

## Available Sensors in Mozambique:

### 1. HC-SR04 Ultrasonic
- **Cost:** ~$2-3 if available
- **Range:** 2cm-400cm
- **Accuracy:** Â±3mm
- **Pros:** Reliable, good range
- **Cons:** Affected by temperature, soft materials

### 2. IR Obstacle Sensor
- **Cost:** ~$1-2
- **Range:** 2cm-30cm
- **Accuracy:** Digital (on/off)
- **Pros:** Cheap, simple interface
- **Cons:** Affected by light/color

### 3. TCRT5000 Line Follower
- **Cost:** ~$1
- **Range:** 1mm-25mm
- **Use:** Could repurpose for close detection

## My Decision:
**Start with 2x HC-SR04 + 1x IR sensor**
- Reason: Balance of cost and capability
- Can upgrade/add more sensors later

## Alternative If Unavailable:
Use single ultrasonic sensor on servo for scanning
- Slower but functional
- Demonstrates creative problem-solving
