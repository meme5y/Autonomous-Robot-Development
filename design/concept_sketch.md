# Robot Concept Sketch

## Overall Design Philosophy
**"Simple, Modular, Upgradable"**

## Chassis Design Options:
### Option 1: Cardboard Prototype
- Pros: Free, easy to modify
- Cons: Not durable, moisture sensitive

### Option 2: Wood Frame
- Pros: Sturdy, easy to work with
- Cons: Requires tools, heavier

### Option 3: Plastic Container
- Pros: Water-resistant, lightweight
- Cons: Need to find suitable container

## Dimensions (Planned):
- Width: 15-20cm (fits through doorways)
- Length: 20-25cm (stable base)
- Height: 10-15cm (low center of gravity)

## Material Sourcing Strategy:
1. Check local electronic repair shops for parts
2. Ask schools/offices for discarded electronics
3. Use household items creatively
4. 3D print only if absolutely necessary
